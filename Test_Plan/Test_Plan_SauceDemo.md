# Plan de Pruebas – SauceDemo

## Objetivo

Validar el correcto funcionamiento de las principales funcionalidades de la plataforma de comercio electrónico SauceDemo mediante pruebas manuales.

## Alcance de las pruebas

Los módulos que serán evaluados son:

- Inicio de sesión (Login)
- Catálogo de productos
- Carrito de compras
- Proceso de compra (Checkout)

## Tipos de pruebas

Durante este proyecto se realizarán los siguientes tipos de pruebas:

- Pruebas funcionales
- Pruebas negativas
- Pruebas exploratorias
- Pruebas de usabilidad

## Entorno de pruebas

Navegador: Google Chrome Versión 145.0.7632.117 (Compilación oficial) (64 bits)  
Sistema operativo: Windows 11 Pro Versión 25H2  
Sitio de pruebas: https://www.saucedemo.com

## Entregables

- Casos de prueba
- Reportes de errores
- Reporte de ejecución de pruebas
- Evidencias de pruebas (capturas de pantalla)
