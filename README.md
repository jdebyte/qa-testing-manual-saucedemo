# Proyecto de Testing Manual – SauceDemo

## Descripción

Este proyecto simula el proceso de aseguramiento de calidad en una plataforma de comercio electrónico.

El objetivo fue diseñar y ejecutar casos de prueba manuales para validar las principales funcionalidades del sistema, identificar defectos y documentar el proceso de pruebas.

Las pruebas fueron realizadas utilizando la plataforma de demostración SauceDemo.

## Objetivos

- Validar las funcionalidades principales de la aplicación
- Identificar posibles defectos en el sistema
- Diseñar casos de prueba estructurados
- Documentar el proceso de pruebas

## Módulos probados

- Login
- Productos
- Carrito de compras
- Checkout

## Herramientas utilizadas

- Pruebas manuales
- Diseño de casos de prueba en Excel
- Reporte de errores
- Documentación en GitHub

## Resultados

Total de casos de prueba: 78  
Casos ejecutados: 78  
Casos exitosos: 71  
Casos fallidos: 7  
Bugs identificados: 7
