ID: BUG-CART-07  
Título: El carrito mantiene productos después de cerrar sesión y volver a iniciar sesión

Prioridad: Media  
Severidad: Media  
Módulo: Carrito de compras  
Entorno: Google Chrome Versión 145.0.7632.117 (64 bits)  

Pasos para reproducir

Iniciar sesión con un usuario válido.  
Agregar uno o más productos al carrito.  
Cerrar sesión.  
Volver a iniciar sesión con el mismo usuario.  
Revisar el carrito de compras.  

Resultado esperado

El carrito debería reiniciarse o vaciarse después de cerrar sesión.

Resultado actual

El carrito mantiene los productos agregados anteriormente después de iniciar sesión nuevamente.
